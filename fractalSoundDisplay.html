In order to load the fractal and play the music, you must be running a locally hosted server. To do this, open up the Terminal (either search for it or look in your Applications folder) and move to the project folder. Then type python -m SimpleHTTPServer 8000 into the terminal and type enter. Then in the search bar of Chrome, type http://localhost:8000/ into the search bar followed by the the file (like index.html) that you would like to see.
